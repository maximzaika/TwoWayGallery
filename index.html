TwoWay Gallery v 2.05

User friendly, easy to use, and touch and navigation enabled JavaScript image gallery.

Download Github Repo

Demo & Configuration

Options that require manual input are not displayed. You may find all available options in Github.

startItem The starting image is changed based on its index (both Main and Secondary galleries).

displayItems The number of items displayed in desktop view.

descriptionType The style of the description.

enableArrowKeys Allows to use arrow keys if in viewport.

enableTouch Enables swiping right and left.

apEnable Enables auto scroll.

apTimeout If apEnable is true, controls auto scroll delay.

apDirection If apEnable is true, controls the direction of auto scroll.

apPauseOnHover If apEnable is true, pauses auto scroll on hover.

apPauseNotif If apEnable is true, notifies the user if auto scroll is paused.

navEnable Displays the navigation arrows.

navShowOnHover Navigation is hidden and shown on hover.

sgEnable Enables Secondary Gallery.

sgInstant Upon click, ignores image scroll animation.

sgDeskTouch Enables mouse drag. Mobile is always on.

sgNavArrows Displays the navigation arrows.
